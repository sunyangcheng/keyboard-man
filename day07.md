### 收获&学习心得&心路历程：

一、滚动

- ctrl + f -> 向下滚动一屏
- ctrl + b -> 向上滚动一屏
- ctrl + d -> 向下滚动半屏
- ctrl + u -> 向上滚动半屏
- ctrl + e -> 向下滚动一行(光标不动)
- ctrl + y -> 向上滚动一行(光标不动)

二、改建

- shift + J -> 5j(向下滚动5行)
- shift + K -> 5k(向上滚动5行)

三、快速定位

- zz -> 将当前行置于屏幕中央
- zt -> 将当前行置于屏幕顶部
- zb -> 将当前行置于屏幕底部
- gg -> 跳到文件首
- G -> 跳到文件尾
- n+gg -> 跳到指定行n
- n+G -> 跳到指定行n
