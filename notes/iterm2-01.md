### 收获&学习心得&心路历程航

目标：掌握导航

1. 分屏

* 左右: cmd + d
* 上下: cmd + shift + d
* 切换分屏：cmd + option + ⽅向键
* 切换分屏：cmd + [/]
* 调整窗口: ctrl + cmd + 方向键

2. 标签

* 新建标签: cmd + t
* 关闭标签: cmd + w
* 切换标签:
  * cmd + 数字
  * cmd + 方向
* 移动标签顺序: cmd + shift + 方向键
